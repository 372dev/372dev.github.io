#### 나의 답안.. (아주 지저분 하군요!)
```java
class Solution {
    public String solution(int n) {
        String answer = "";
        
        if (n < 1) {
            return "";
        }
        
        if (n % 2 == 0) {
            for(int i = 0; i < n/2; i++) {
                answer += "수박";
            }
        } else {
            answer += "수";
            if (n > 1) {
                for(int i = 0; i < (n/2); i++) {
                answer += "박수";
            }
            }
        }
        
        return answer;
    }
}
```
