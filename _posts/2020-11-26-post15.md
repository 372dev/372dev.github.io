#### 상위 N개 레코드
![ORACLE_LEVEL1_10](https://raw.githubusercontent.com/372dev/372dev.github.io/master/_posts/imgs/ORACLE_LEVEL1_10.PNG)
```ORACLE
SELECT NAME
FROM (
    SELECT RANK() OVER(ORDER BY DATETIME ASC) "RANK", NAME
    FROM ANIMAL_INS
    )
WHERE RANK = 1;
```
